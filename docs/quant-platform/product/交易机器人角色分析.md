# 交易机器人角色分析 — 现状与差距

**日期**: 2026-05-11  
**核心问题**: 系统是否有一个交易机器人角色来分析市场信息、综合因子/新闻/宏观、排名并输出交易策略？

---

AI Agent **已经能做**交易决策。刚才问它 "你是量化交易机器人，给出BTC交易决策"：

```
决策: SELL (-10分)
```

**但它和 Strategy 之间没有连接。** Strategy 还在用两行硬编码规则。

## 二、差距总结

| 维度 | 当前 | 需要的 |
|------|------|--------|
| **数据输入** | 只用 composite_score + rank | 因子(25个) + 新闻 + 宏观 + 价格 |
| **分析角色** | 无 — 机械规则 | AI 交易员角色 (System Prompt 定义) |
| **排名逻辑** | rank_in_date (DB预计算) | AI 综合排名 (因子×宏观×新闻) |
| **决策输出** | enter_long True/False | {action, confidence, sl, tp, reasoning} |
| **连接方式** | Strategy 不调 AI | Strategy → AI Agent → 决策 → 执行 |
| **可解释性** | 无 | AI 给出每笔交易的推理链 |

## 三、解决方案：AI 交易员角色

### 3.1 角色定义

```yaml
name: ai-trader
role: |
  你是一个专业的加密货币量化交易员。你的职责是:
  1. 综合分析因子信号、新闻情绪、宏观经济
  2. 对候选币对进行排名
  3. 输出结构化的交易决策

decision_rules:
  - 必须基于数据, 不得猜测
  - 给出 buy/sell/hold 明确方向
  - 附带置信度 (0-1) 和推理链
  - 考虑风险: 止损位、仓位比例
```

### 3.3 实现方案

**方案A: Strategy 内嵌 AI 调用** (推荐)

```python
class AITraderStrategy(IStrategy):
    def populate_entry_trend(self, df, metadata):
        pair = metadata['pair']

        # 1. 基础因子过滤 (快速)
        factor_ok = df['composite_score'] > 0.2

        # 2. AI 交易员决策 (每小时调一次)
        if self._should_call_ai():
            decision = self._ask_ai_trader(pair)
            self._ai_cache[pair] = decision

        # 3. 综合判断
        ai = self._ai_cache.get(pair, {})
        return factor_ok & (ai.get('action') == 'buy') & (ai.get('confidence', 0) > 0.5)

    def _ask_ai_trader(self, pair):
        """调用 AI Agent 的 /quant/chat 端点"""
        resp = requests.post(f"{API}/quant/chat", json={
            "messages": [{
                "role": "system",
                "content": AI_TRADER_SYSTEM_PROMPT
            }, {
                "role": "user",
                "content": f"分析 {pair} 并给出交易决策"
            }],
            "skill": "risk-assessment"  # 使用 Skills 系统
        })
        return self._parse_decision(resp.json()['response'])
```

**方案B: 独立决策服务**

```
独立的 AI Trader Service
  ├─ 每小时运行
  ├─ 对所有币对分析排名
```

**推荐方案A** — 更简单，保持 Strategy 自包含。

## 五、结论

**问题**: 系统是否有一个交易机器人角色？

**答案**: 
- **有分析能力** — AI Chat Agent 可以对任何币对做"因子+新闻+宏观"综合分析并给出交易建议
- **有执行框架** — freqtrade Strategy 可以自动执行买卖
- **但两者没有连接** — Strategy 用的是两行硬编码规则，AI Agent 的决策没有被 Strategy 调用

**只需要做一件事**: 让 Strategy 调用 AI Agent 的决策，即可形成完整的 "AI 交易机器人"。
